# Chatify

Chatify is a real-time chat application built using the MERN stack and WebSockets.  
It allows users to communicate instantly through a simple and responsive web interface.

## What it does

- Real-time one-to-one and/or group messaging
- Instant message delivery using WebSockets
- Persistent chat sessions through a backend server

## Tech Stack

- JavaScript
- MERN Stack (MongoDB, Express, React, Node.js)
- WebSockets

## Purpose

This project was built to understand and implement real-time communication on the web using WebSockets and a full-stack JavaScript setup.  
It focuses on speed, reliability, and simplicity.

The application is functional and supports basic real-time chatting.
